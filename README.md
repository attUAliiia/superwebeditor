# superwebeditor

a simple chrome extension that lets you edit text on any website and keeps your changes even after you refresh the page.

### features
- **visual editing**: just point and click to edit text.
- **persistent**: saves your changes to chrome's local storage so they stay there.
- **easy controls**: toggle on/off from the menu or use `esc` to quit.
- **visual cues**: green border when hovering, red border when editing.

### how to install
1. download or save all the files into a folder.
2. open chrome and go to `chrome://extensions`.
3. turn on **developer mode** (top right switch).
4. click **load unpacked** (top left).
5. select the folder where you saved the files.

### how to use
1. click the extension icon in your toolbar.
2. click the **enable editing** button.
3. hover over any text (it will show a green border).
4. click the text to start typing (border turns red).
5. click anywhere else to save your changes.
6. press **esc** to stop editing instantly.

### notes
- changes are saved locally on your computer.
- might not work perfectly on super complex sites like facebook or gmail (dynamic content), but works great on blogs, wikis, and static pages.
